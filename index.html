Google Sheets Integration

Setup Instructions:

  1. Create a new Google Sheet
  2. Share it publicly: Share → "Anyone with the link" → Viewer
  3. Get an API key from Google Cloud Console
  4. Enable the Google Sheets API for your project
  5. Click "Connect to Google Sheets" below and enter:
    • Your API key
    • Your complete Google Sheets URL (copy from your browser)
